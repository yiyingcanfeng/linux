# 新增 Chromium OS 镜像

2016-12-27 Yiqun Hui

TUNA 镜像站新增chromium OS源代码镜像。

由于各方面原因，该镜像暂时只镜像master分支，即只有出现在这个文件中的项目才会被同步。

使用方法见 chromium OS 镜像使用帮助。
